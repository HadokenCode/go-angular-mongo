What is GoGAM?

GoGam is stands for:

  • Go (GoLang)
  • G (Gin Gonic)
  • A (AngularJS 1.x)
  • M (MongoDB)

This is a CRUD sample built with GoLang + Gin + Angular + MongoDB stack, this can be use for web application stack.
